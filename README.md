# Config
```
Flavor: "&lVisit our store at"
ShopLink: "&b&lstore.spasboy.tk"
```

# Permission Nodes
```yml
  shop:
    description: Sends the shop link to the player
    usage: /<command>
    permission: shopquick.use
  reload:
    description: Reloads the Config
    usage: /<command>
    permission: shopquick.reload
```
